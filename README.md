# Usage

    mkdir -p ~/dev/
    cd ~/dev/
    git clone git@github.com:vorburger/vorburger-dotfiles-bin-etc
    cd vorburger-dotfiles-bin-etc

    ./dnf-install.sh
    ./symlink.sh
    ./gnome-settings.sh

NB: The `~/dev/vorburger-dotfiles-bin-etc/` path is currently hard-coded e.g. in `dotfiles/bashrc`.
